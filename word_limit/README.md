# Word-Limit

Permits you to limit the number of words in some text

## Usage

### `{exp:word_limit}`

#### Example Usage

```
{exp:word_limit total="100"}
	text you want processed
{/exp:word_limit}
```

#### Parameters

- `total` - lets you specify the number of words.
